# 内容管理 - 帖子删除/置顶、回复管理

## Goal

实现内容管理功能：帖子删除/置顶、回复管理

## Requirements

### 1. 帖子管理
- [ ] 后端：帖子删除 API（含权限校验：管理员可删除任何帖子，普通用户只能删除自己的）
- [ ] 后端：帖子置顶/取消置顶 API
- [ ] 前端：帖子列表显示置顶状态
- [ ] 前端：删除按钮（根据权限显示）
- [ ] 前端：置顶/取消置顶操作

### 2. 回复管理
- [ ] 后端：回复删除 API（含权限校验）
- [ ] 前端：回复列表显示删除按钮（根据权限显示）

### 3. 权限规则
| 角色 | 帖子删除 | 帖子置顶 | 回复删除 |
|------|----------|----------|----------|
| 管理员 | 可删除任何人的帖子 | 可置顶任何帖子 | 可删除任何人的回复 |
| 普通用户 | 只能删除自己的帖子 | 无 | 只能删除自己的回复 |

## Technical Notes

### 需要新增的 API
- `POST /api/forum/admin/post/delete` - 删除帖子
- `POST /api/forum/admin/post/pin` - 置顶/取消置顶帖子
- `POST /api/forum/admin/reply/delete` - 删除回复

### 需要修改的现有文件
- `Forum_Post` 模型已有 `isPinned` 字段
- 需要添加权限校验

## Out of Scope

- 批量操作
- 操作日志（本任务先不做，待单独任务）
