# 帖子详情页优化增强

## Goal
优化帖子详情页，添加侧边栏（作者信息、相关帖子、社区准则）和互动功能（点赞、收藏、分享），提升用户体验。

## Requirements

### 1. 帖子详情页 UI 优化
- 采用双栏布局（内容区 70% + 侧边栏 30%）
- 添加面包屑导航（返回 > 分类 > 标签）
- 统一主色调为 `indigo-600`（以首页为准）

### 2. 侧边栏组件
- **作者信息卡片**：显示作者头像、昵称、职位、简介、"关注"按钮
- **相关帖子推荐**：显示同分类或同标签的热门帖子
- **社区准则**：显示社区规则卡片

### 3. 互动功能
- **点赞功能**：用户可以对帖子点赞，支持取消点赞
- **收藏功能**：用户可以收藏帖子，支持取消收藏
- **分享功能**：生成分享链接

### 4. 后端 API 支持
- 点赞/取消点赞 API
- 收藏/取消收藏 API
- 获取帖子互动状态 API（是否点赞、是否收藏）
- 获取相关帖子 API
- 分享统计 API

## Acceptance Criteria
- [ ] 后端 API 支持点赞、收藏、分享相关功能
- [ ] 帖子详情页采用双栏布局
- [ ] 侧边栏显示作者信息卡片
- [ ] 侧边栏显示相关帖子推荐
- [ ] 侧边栏显示社区准则
- [ ] 面包屑导航功能正常
- [ ] 点赞/收藏按钮状态正确显示

## Technical Notes

### 数据库
- 需要新增 `Forum_Collection` 收藏表（如果不存在）
- Post 模型已有 `likeCount` 字段

### API 设计
```
POST /api/forum/like          - 点赞/取消点赞
POST /api/forum/collect       - 收藏/取消收藏
GET  /api/forum/post/{id}/status  - 获取互动状态
GET  /api/forum/post/{id}/related  - 获取相关帖子
POST /api/forum/post/{id}/share    - 分享统计
```

### 前端技术栈
- React + TypeScript
- TailwindCSS（主色调 indigo-600）
- 组件化开发
